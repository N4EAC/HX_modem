# HX_modem v0.8.5
HX modem is my first radio modem project.
I have't been happy with the performance in a real HF frequency environment.
The software still lacks development. 
So for the current state, I have CAT control for Yaesu FT-710 and Kenwood TS-2000.
The modem protocol communicated well inside audio cables.
The issue is not decoding with added noise and other sounds. I managed to decode HX in the middle of FT8 transmissions.
The problem was decoding radio to radio. Even with S3 signal I had CRC errors and sometimes no decode.
So I am leaving this open here if anyone wants to take a chance to test and further develop HX.
HX stands for Hyper eXchange in my initial concept.
There are two modem speeds. F (fast) N (normal).

The version in this repository has also a TS-2000 CAT control which I tested for Thetis.
